Ce projet a été réalisé par Axel Aiello, Matthieu Barzellino et Loïc Dalian.
 
Projet réalisé dans le cadre du Parcours IHM de Décembre 2017 à Mars 2018, sur table tactile et appareil Android.

Alliances et Trahisons est un jeu de conquête galactique dont le but est d’acquérir la majorité de planètes en créant et déplaçant ses vaisseaux. 
Le jeu pousse les joueurs à proposer des alliances  entre eux par des échanges de ressources, puis se trahir pour remporter la victoire. 
Pour gagner il vous faudra jongler entre collaboration et compétition afin de savoir quand retourner votre veste !
